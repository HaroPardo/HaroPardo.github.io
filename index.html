Sistema de Autenticación Empresarial

¿Por qué lo creé?

Identifiqué la creciente demanda de soluciones que utilicen Oracle y Docker, tecnologías empresariales robustas que deseaba dominar. Oracle ofrece un rendimiento excepcional para gestión de datos transaccionales, mientras que Docker proporciona un ecosistema de contenedores ideal para despliegues consistentes y escalables. Decidí combinar ambas tecnologías para crear una solución de autenticación segura y portable.

¿Qué aprendí de este proyecto?

Adquirí experiencia práctica con Oracle Database, comprendiendo su arquitectura y optimizaciones para entornos productivos. Dominé la contenerización con Docker, gestionando imágenes, volúmenes y networking. Aprendí a integrar librerías JDBC y implementar encriptación SHA-256 para seguridad de contraseñas. Además, fortalecí mis habilidades en administración de sistemas Linux, trabajando extensamente con la terminal para gestionar los contenedores y la base de datos.

¿Te gustaría seguir trabajando en proyectos similares?

Definitivamente. Este proyecto me ha permitido salir de mi zona de confort y explorar tecnologías empresariales ampliamente utilizadas en el sector. Me ha motivado a seguir aprendiendo sobre arquitecturas escalables, seguridad de datos y DevOps, áreas cruciales para el desarrollo de aplicaciones modernas y robustas.

Aplicación Android

¿Por qué lo creé?

Siempre he tenido un gran interés en el desarrollo de aplicaciones para Android, dado que son herramientas que utilizamos diariamente. Decidí embarcarme en este proyecto para crear una aplicación útil que pudiera integrarse en la rutina cotidiana, aprovechando la oportunidad para explorar el ecosistema móvil.

¿Qué aprendí de este proyecto?

Aunque ya contaba con experiencia en Java, descubrí que el desarrollo en Android Studio presenta un entorno y configuración únicos. Logré adaptarme y dominar aspectos clave como la integración de APIs de Google, gestión de bases de datos SQLite, y el uso de librerías como OSMDroid para mapas y Picasso para la carga de imágenes.

¿Te gustaría seguir trabajando en proyectos similares?

Definitivamente. Este proyecto ha despertado mi interés en el desarrollo móvil, y estoy ansioso por seguir explorando las particularidades y capacidades únicas que ofrece la plataforma Android, así como las diferencias arquitectónicas y de UX respecto al desarrollo web y de escritorio.

WhatsApp Extension

¿Por qué lo creé?

Al cambiar de navegador web a uno que se me recomendaba por ser más rápido noté rápidamente que no había ningún tipo de extensión de navegador para acceder a WhatsApp, siendo la única manera teniéndolo en una pestaña. Así que se me ocurrió no solo que sea para un navegador, sino que lo puedas tener siempre abierto, ideal para trabajar para poder pasar archivos del móvil al ordenador y viceversa o incluso para mero ocio.

¿Qué aprendí de este proyecto?

Aprendí a usar el framework Electron y a usar CommonJS, además de elementos que ya había usado como CSS, JavaScript o HTML. Tuve nuevos retos con ellos aprendiendo a usarlos en aplicación de escritorio, ya que su uso anterior a excepción de CSS fue exclusivamente en páginas web.

¿Te gustaría seguir trabajando en proyectos similares?

Sí, además, tengo pensado seguir sacando actualizaciones en esta aplicación hasta poder perfeccionarla e intentar añadir más funcionalidades.

El aprendizaje más significativo fue confirmar que con dedicación e investigación puedo resolver cualquier desafío técnico.

¿Te gustaría seguir trabajando en proyectos similares?

Definitivamente. Los proyectos complejos que requieren meses de desarrollo ofrecen el mayor crecimiento profesional. Estoy entusiasmado por seguir explorando el potencial de Java y otras tecnologías para crear soluciones robustas con impacto positivo.

API de Base de Datos de Películas

¿Por qué lo creé?

Desarrollé esta API REST como plataforma para profundizar en la gestión profesional de bases de datos SQL. Partiendo del concepto simple de un repositorio de películas, expandí el proyecto para explorar migraciones entre sistemas de gestión de bases de datos (de MySQL a PostgreSQL) y su implementación en entornos en la nube como Azure.

¿Qué aprendí de este proyecto?

Consolidé mi comprensión de las diferencias entre operaciones REST y CRUD, dominé el diseño avanzado de esquemas relacionales y adquirí experiencia práctica en administración de bases de datos. Este proyecto complementó mi experiencia previa con MongoDB profundizando en sistemas SQL tradicionales.

¿Te gustaría seguir trabajando en proyectos similares?

Sí, las bases de datos son componentes críticos en cualquier sistema moderno. Estoy comprometido a seguir especializándome en este campo, particularmente en optimización de consultas, diseño de esquemas escalables e implementación en la nube.
