AWS Portfolio Monitoring System

Arquitectura Serverless

Sistema completo de monitorización automática para mi portfolio web (HaroPardo.github.io) utilizando servicios AWS serverless. La arquitectura elimina la necesidad de servidores dedicados y escala automáticamente.

Flujo de Datos

Detección (Route 53): Health check externo que verifica https://haropardo.github.io cada 30 segundos desde servidores AWS globales.
Evaluación (CloudWatch Alarm): Métrica HealthCheckStatus que activa alarma solo tras 3 fallos consecutivos, evitando falsos positivos.
Disparador (EventBridge): Regla que escucha cambios de estado de alarma (OK → ALARM o ALARM → OK).
Procesamiento (Lambda): Función Python que formatea mensajes y publica en SNS.
Entrega (SNS): Tema que reenvía notificaciones a suscriptores (mi email).

Beneficios y Aprendizaje

Este proyecto demostró mi capacidad para diseñar e implementar arquitecturas cloud nativas. Aprendí a orquestar múltiples servicios AWS, gestionar permisos IAM, y crear sistemas serverless completamente automatizados. La solución es costo-eficiente (solo pago por uso) y altamente disponible.
